Educational Background

I received an entrance scholarship to the value of R20 000 and I also got awarded a bursary by Studietrust to study up until honours with all expenses taken care of.

This meant that I only had to worry aout my grades and nothing else. In my first year, I learnt how to write code in Python, Java and SQL with intermediate insertion, deletion and search queries. I created a checkers game in the same year using Java's swing librsry for the GUI.

This programming course equipped me with the necessary skills to create interactive websites like the one you are reading. I learnt HTML, CSS, JavaScript, SQL and some terminal. I even have a certificate to show for it and a reference included.
